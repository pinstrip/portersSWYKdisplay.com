Introduction

programing is were you type in step by step commands for your computer to follow. however computers don't understand normal language, these commands have to be written in a programing language. there is more than one programming language, and i have used a few. here are the ones i have used and a bit of info about them

JavaScript();

JS is a core part of the web, it is used in 99% of webpages! it was created in 1995 to allow websites to be interactive instead of just being boring ol text.

Html

Html is what you use to actually program websites (like this one). it is used by all web browsers. it was orignaly created so CERN(European Organization for Nuclear Research) researchers could share documents.

Python()

python is a coding language that is aimed at beginners. it has very basic syntax and has been criticized for its slow speed and heavy memory usage.

C# ();

C# is a simple, general purpose programing language that is used by many companies to build applications. it was originally created as a version of the C programing language that supported object-oriented programing.

C++();

C++ is a high level programing language that influenced almost every item on this list. it was created in 1982 as a successor to C.
